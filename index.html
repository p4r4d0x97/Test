FILTER & SEARCH HELP
Type any of these patterns into the search box

BASIC SEARCH

10.93.1 — finds anything containing this text in IP, name, or MAC
printer — substring match (case-insensitive)
C026302 — finds the device with this hostname

FIELD-SPECIFIC FILTERS

vlan=1021 — exact VLAN match
type=printer — by device type
switch=192.168.8.10 — devices on this switch
port=1/32 — specific switch port
status=offline — by ping status (online/offline/no_info)
name=C026 — substring of device name
designation=server — designation field
model=lenovo — model field
serial=PF12345 — serial number
rack=Rack09 — by rack assignment
previous-mac=40:B0 — replaced devices
xml_change_status=new — newly discovered

IP RANGES

ip=10.93.1.10-50 — IP range from .10 to .50
ip=10.93.1.0/24 — entire subnet (CIDR)

REGEX PATTERNS

Auto-detected when value contains |^$.*+?()[]{}\\
name=^C0[12] — names starting with C01 or C02
ip=10\\.93\\.(1|2|3)\\. — IPs in three subnets
name=printer.*hp — printer hostnames containing "hp"

COMBINE WITH SPACE = AND

vlan=1021 status=offline — VLAN 1021 AND offline
type=switch rack=Rack01 — switches in Rack01

NEGATION WITH !

!vlan=1010 — exclude all VLAN 1010 devices
type=pc !status=online — PCs that are NOT online
!name=test — exclude anything with "test" in name

PRACTICAL EXAMPLES

type=printer status=offline — find broken printers
vlan=1010 status=offline — APs that are down
switch=192.168.8.10 !status=online — non-online devices on switch X
ip=10.93.1.10-50 type=pc — PCs in IP range
previous-mac=.+ — all devices that have a previous MAC
